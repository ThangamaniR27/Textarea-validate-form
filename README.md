# Textarea Validate Form

This project is a simple **Textarea Form with Real-Time Character Counter** built using **HTML, CSS, and JavaScript**.

## 📌 Features

- Textarea input field
- Real-time character counter
- Maximum character limit (200 characters)
- Responsive and centered layout
- Simple and clean UI

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
